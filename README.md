# EasemobRestTool
通过curl，实现快速调用环信rest接口

## 使用

* 下载项目
`git clone https://github.com/Mr-Dead-Rabbit/EasemobRestTools.git`

* 配置环信相关信息，如: `get_user.sh`
```
APPKEY="用户自己的appkey"
client_id="用户自己的client_id"
client_secret="用户自己的client_secret"
```

* 运行脚本，输入需要查询的参数
